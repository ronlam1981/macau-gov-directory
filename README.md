# 澳門特區政府部門通訊錄

可搜尋的澳門特別行政區政府部門聯絡資料目錄，資料來源為 [gov.mo](https://www.gov.mo) 官方公開數據。

**網站：** https://ronlam1981.github.io/macau-gov-directory/

---

## 更新資料

雙擊 `發佈更新到GitHub.command` 即可一鍵完成所有操作：

1. 從 gov.mo 下載最新的政府部門及諮詢組織通訊錄
2. 與現有資料逐項對比（部門名稱、領導人、電話、地址等所有欄位）
3. 顯示變更摘要（新增/移除部門、領導人變動、其他資料變更）
4. 自動更新 index.html 並推送到 GitHub Pages

全程自動，無需手動確認。如線上資料與現有資料完全一致，會顯示「無需更新」並結束。

### 首次使用前置準備

1. 確保已安裝 [Python 3](https://www.python.org)
2. 確保已安裝 [Git](https://git-scm.com)
3. 首次執行時，腳本會自動安裝所需 Python 套件（pandas、openpyxl）
4. 首次執行時，腳本會引導你登入 GitHub 並建立遠端倉庫

### 檔案說明

| 檔案 | 說明 |
|---|---|
| `發佈更新到GitHub.command` | 主程序入口 — 雙擊執行，一鍵更新及發佈 |
| `updatedata.py` | 核心更新邏輯 — 下載、對比、生成新資料 |
| `index.html` | 網站頁面 — 包含所有部門資料及前端介面 |
| `Rawdata/` | 從 gov.mo 下載的原始 Excel 檔案 |
| `.github/workflows/` | GitHub Actions 自動部署設定 |

## 網站功能

- **即時搜尋：** 輸入關鍵字即時篩選部門名稱
- **分類篩選：** 按司級分類瀏覽（行政長官、行政法務司、經濟財政司等共 14 個分類）
- **部門詳情：** 顯示地址、電話、傳真、電郵、網址
- **領導人資料：** 顯示部門主管及其職銜
- **行動裝置適配：** 支援手機及平板瀏覽
- **單檔離線可用：** 所有資料內嵌於 index.html，無需伺服器

## 資料涵蓋範圍

涵蓋澳門特區政府 338 個部門及機構，包括：

- 行政長官及轄下部門
- 五個司級部門（行政法務司、經濟財政司、保安司、社會文化司、運輸工務司）
- 廉政公署、審計署
- 立法機關、司法機關
- 功能組織、諮詢組織
- 公共事業機構
